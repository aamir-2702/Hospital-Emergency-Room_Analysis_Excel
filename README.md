# Hospital-Emergency-Room_Analysis_Excel
I recently developed an Interactive Hospital Emergency Room Dashboard using Microsoft Excel to analyze patient flow and operational performance in the emergency department.

## 📌 Project Overview
This project presents an **Hospital Emergency Room Dashboard** created using **Microsoft Excel**.  
The goal of this project is to analyze hospital emergency room data and provide insights into patient flow, wait times, and service performance.

The dashboard helps visualize important healthcare metrics that can support better decision-making and improve hospital service efficiency.


---


## Business Problem
We need to create a Hospital Emergency Room Analysis Dashboard in Excel to improve efficiency and provide useful insights. This dashboard will help stakeholders monitor, analyze, and make better decisions for managing patients and improving services.

- Analyze daily emergency room patient visits
- Monitor average waiting time trends
- Track patient satisfaction scores
- Identify admission vs non-admission patterns
- Create an interactive dashboard for quick insights

## 🎯 Key KPIs
The dashboard focuses on the following key performance indicators:

- **Total Number of Patients** – Count of patients visiting the emergency room.
- **Average Wait Time** – Average time patients wait before seeing a medical professional.
- **Patient Satisfaction Score** – Average satisfaction rating from patients.
- **Admission Status** – Number of patients admitted vs not admitted.
- **Timeliness** – Percentage of patients attended within 30 minutes.
- **Age Distribution** – Number of patients by different age groups.
- **Gender Analysis** – Male vs female patient distribution.
- **Department Referrals** – Departments where patients are referred most frequently.

## 🧠 Key Insights
- Most ER patients fall between **20–49 years age group**.
- Around **60% of patients are attended within the target time**, while **40% experience delays**.
- **General Practice and Orthopedics** receive the highest patient referrals.
- Patient satisfaction remains high with an average score of **4.91**.
